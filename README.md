# USV_Analysis
for analyzing USVs
This is a general workflow to analyze usv data from rats that has been preprocessed on deepsqueak

# data output convention for deepsqueak:
1. deepsqueak: this folder has the networks and the clustering models when you do your detection algorithm
2. logs: has the logfiles when you do your clusters
3. det: det are the detection files, they are matfiles that contain a table
4. training: the training network files
5. detections: 
6. extracted contours: these are the contours that accompany each cluster
